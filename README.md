# Tulip Flower Shop 🌷  E-Commerce Storefront

> A full e-commerce storefront for a flower shop with CMS-managed inventory, shopping cart, and a clean editorial design — built with Next.js, TypeScript  and Tailwind CSS.

## 📌 What This Does

Tulip is a production e-commerce web application for a flower shop. The shop owner manages all products, categories, and content through Sanity Studio (a headless CMS), while customers browse products, view details, and add items to a cart — all rendered with Next.js for fast page loads.

---

## ✨ Features

- 🛒 Shopping cart with add/remove/quantity controls
- 🌸 Product listings with category filtering
- 📝 CMS-managed products, descriptions, and images via Sanity
- ⚡ Static generation with Next.js for fast load times
- 📱 Fully responsive design with Tailwind CSS

## 🚀 Getting Started

```bash
git clone https://github.com/EmanIqbal620/tulip-E-commerce-web
cd tulip-E-commerce-web
npm install
```

### Environment Variables

```env
NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id
NEXT_PUBLIC_SANITY_DATASET=production
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Sanity Studio

```bash
cd sanity-studio
npm install
npm run dev
```
